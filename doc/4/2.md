# Blender

Блендер - это комбинация множества инструментов. Это:

* Modeler - для создания 3D моделей
* рендерер - для создания фотореалистичных визуализаций 3D-моделей
* rigger - для создания «ригов», которые представляют собой специально разработанные 3D-модели для анимации.
* система анимации - для создания анимации по ключевым кадрам
* композитор - для объединения изображений из разных источников для создания видеокадров
* средство визуализации видео - для вывода полной анимации в различных видеоформатах

Для этих руководств по WebGL мы будем использовать только функции моделирования Blender. Вы не освоите Blender за короткое время. Чтобы стать «опытным пользователем» Blender, нужны годы.

Чтобы ощутить всю мощь Blender, посмотрите одну из анимаций, размещенных на YouTube-сайте Blender Foundation.

Вам необходимо установить Blender на свой компьютер. Чтобы избежать проблем с прокси-сервером и сканированием, загрузите установщик из папки K: \ DF \ DFCS \ CS474.

ВАЖНО: для эффективной работы с Blender вы должны использовать трехкнопочную мышь.

## Some Preliminary Issues

Blender использует правостороннюю систему координат, но другую ориентацию представления по умолчанию по сравнению с WebGL.

* В Blender плоскость земли - это плоскость X-Y, а ось Z направлена вверх.
* В WebGL плоскость земли - это плоскость X-Z, а ось Y направлена вверх.

При экспорте Blender модели средство экспорта может изменить ориентацию в соответствии с ориентацией WebGL. Пример ниже показывает ориентацию вида Blender.

Создавая модели для программ WebGL, вы можете сохранить несколько моделей в одном файле или разделить модели в отдельные файлы. Если ваши модели находятся в отдельных файлах, убедитесь, что вы используете согласованные единицы измерения, чтобы модели «подходили друг к другу» по мере необходимости.

Каждая модель требует уникального имени, чтобы ею можно было управлять отдельно в ваших программах WebGL. НЕ включайте пробелы в имена моделей, потому что имена моделей станут именами свойств объектов в ваших программах JavaScript WebGL, а в именах свойств не должно быть пробелов.
