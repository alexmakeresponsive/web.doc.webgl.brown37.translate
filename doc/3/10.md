# Lines

WebGL поддерживает отображение линий. Это полезно, если вы хотите нарисовать каркасную визуализацию вашей модели или очертить лицо рамкой или выделением.

В WebGL вы всегда определяете одну или несколько линий, используя массив вершин. Есть три различных способа использования вершин для формирования линий, как показано на диаграмме ниже. Возможные варианты:

* `LINE` - две вершины для каждой линии. Если вершина нужна для нескольких строк, ее нужно повторить в массиве вершин. Для определения n линий требуется 2n вершин.
* `LINE_STRIP` - после двух начальных вершин каждая дополнительная вершина определяет еще одну линию. Для определения n строк требуется (n + 1) вершин.
* `LINE_LOOP` - идентично LINE_STRIP с добавлением линии, соединяющей первую и последнюю вершины. Для определения n линий требуется n вершин.
