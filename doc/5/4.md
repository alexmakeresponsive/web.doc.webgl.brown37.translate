# A Simple Model

Визуализация модели - непростой процесс. Будет легче понять, как все работает, если мы начнем с простейшего возможного примера и постепенно добавим сложности. Когда вы визуализируете сложные модели, вы можете создавать эти модели в Blender (или другом инструменте моделирования). Для наших целей давайте вручную определим очень простую 3D-модель для работы.


## A Simple 3D Model

Изучите код в файле «simple_model_01.js» ниже. Обратите внимание на следующие важные идеи:

* Функция `Triangle` (строка 37) - это определение класса, которое не содержит никаких функций. Он будет содержать данные, определяющие один треугольник. Для начала, объект Triangle будет содержать массив из трех вершин.
* Функция `SimpleModel` (строка 48) - это определение класса, которое не содержит никаких функций. Он будет содержать «имя» модели и массив треугольных объектов. Модель в простейшей форме - это просто набор треугольников.
* Функция `CreatePyramid` (строка 59) создает экземпляр класса SimpleModel с именем model, дает ему имя, называемое «простой», и устанавливает его массив треугольников в 4 треугольника пирамиды.

> Вы можете отредактировать код JavaScript и «перезапустить» рендеринг, если хотите поэкспериментировать. Если вы вводите ошибки в коде, вам, возможно, придется перезагрузить страницу, чтобы получить свежую, безошибочную версию кода.

В следующей последовательности уроков мы будем использовать эту простую модель, чтобы делать следующее:

* Нарисуйте пирамиду, используя один цвет для всей модели.
* Нарисуйте пирамиду, используя разные цвета для каждой грани.
* Визуализируйте пирамиду, используя разные цвета для каждой вершины.
* Визуализируйте пирамиду, используя координаты текстуры для каждой вершины.
* По мере изучения этих различных примеров вы обнаружите, что ваши программы шейдеров, буферные объекты и код рендеринга JavaScript очень взаимозависимы. Когда вы меняете один из них, все они должны измениться!


## A Side Note about JavaScript Functions

Чтобы демонстрационный код можно было редактировать на веб-странице, функции должны быть определены с использованием нестандартного синтаксиса, который делает их изменяемыми «постфактум». Функции добавляются в среду JavaScript при загрузке веб-страницы. Но демонстрационный код должен их переопределить.
